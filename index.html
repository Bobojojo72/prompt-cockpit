🚀 How to Climb All 4 Levels of Claude in One Weekend

1 Claude Chat · Saturday AM

Go to claude.com/download. Install the app.

Pay the $20. Select Opus 4.6 + Extended Thinking.

Connect Slack, Drive, Notion through Connectors.

Stop writing long prompts. Prompt this instead:

"I want to [TASK] for [SUCCESS CRITERIA]. Use AskUserQuestion before you start."

Most people stay here forever. You're leaving 90% of Claude untouched. → how-to-claude.ai

2 Claude Cowork · Saturday PM

Go to Cowork. Create a folder "Claude-Cowork."

4 subfolders: About me, template, project & outputs.

Create about-me.md: what you do, how you do it.

Create anti-ai-style.md: words you'd never say.

Set Global Instructions (Settings → Cowork → Edit):

"Always read my files first, never edit my originals, deliver everything to CLAUDE OUTPUTS."

You just killed prompting. From now on: 2 lines + your folder. → claude-co.work

3 Skills + Plugins · Sunday AM

Open Cowork. Type: "Use the skill-creator to help me build a skill for [your most repeated task]."

Claude interviews you. Answer and be specific.

Test it: "When would you use this skill?" Fix if vague.

Upload: Settings → Capabilities → Skills → Upload.

Now it fires automatically. No slash command.

Install Plugins: Cowork > Customize > Browse plugin.

Skills with about-me.md = two layers simultaneously. → ruben.substack.com/p/claude-skills

4 Code + Computer · Sunday PM

Click the Code tab. Connect GitHub (free account → Settings → Connectors).

"Create a GitHub repo named [project]. Code everything. Don't ask for permissions."

Download VS Code. Install Claude extension. Turn on "Skip Permissions" for 100x speed.

After your session, paste:

"Create a CLAUDE.md file with everything you learned about this project."

Computer use: Settings → Desktop app → turn on Browser use + Computer use.

Connect your phone with Dispatch. Text a task.

Schedule a recurring task: left sidebar → Scheduled → write the prompt → pick the frequency.

Claude code guide: ruben.substack.com/p/claude-code · Claude Computer: ruben.substack.com/p/claude-computer

This is where you stop working inside Claude. Claude starts working on your computer.

🤖 12 Main Ways to Use AI — Master Playbook

1 PROMPT ENGINEER

The foundation. Always use the 5 steps: Expert → Task → Context → Constraints → Ask. The clarifying questions step reveals gaps in your own thinking.

2 SPARRING PARTNER

Use when you have an idea and need it challenged. Find holes in your plan, identify weak assumptions. NOT a cheerleader — a critical investor or devil's advocate.

3 24/7 TUTOR

Use when you want to learn something. Take screenshots, paste errors, describe what you see. Ask it to walk you through things step by step.

4 SKILLS

Break your repeatable tasks into skills (pre-built prompts/workflows). In Claude Cowork: skills like /research, /write, /edit, /post are reusable.

5 MEMORY

Give AI your context ONCE so it remembers it every time. In Claude: this is your claude.md or memory files. Keep updating it as your project evolves.

6 REPETITION

In long conversations, AI can drift. Remind it of what matters. In Claude Code: run /q at start of session. Paste a reminder of your top priorities.

7 PLANNING

Use plan mode for big tasks. Review the plan before execution. In Claude Code: SHIFT+TAB to toggle plan mode. Build → review → approve → execute.

8 MCP

MCP = Model Context Protocol. Connects AI to your actual apps and data. Claude can then act in Google Sheets, Gmail, Slack, Airtable etc for you directly.

9 STACKING

Combine Skills + MCP. A skill tells MCP what to do and it executes it. Example: /post skill → triggers LinkedIn MCP → schedules and posts your content.

10 CONTEXT ENGINEERING

Evolved from prompt engineering. Give AI the full picture of your world: product, revenue, user base, struggles, runway. Richer context = expert-level output.

11 AI SYSTEM

Build a system that runs automatically. Example: customer support bot that reads tickets, searches your knowledge base, and drafts responses. Connects to CRM, billing, booking.

12 DOCUMENTATION

Your AI brain. A living document that describes: what your company does, your products, your problems, your style, your goals. Formula: AI Leverage = Skill × Clarity.

🧠 AI Tips & Mindset — 8 Key Principles

🤨

1. SKEPTICISM

Especially with AI influencers — question everything. Not all AI advice is equal. Think critically, test things yourself before sharing.

❤️

2. LOVE LEARNING

AI is evolving by the hour. Stay curious. Play around with everything — the learning IS the work right now.

📢

3. LEARN IN PUBLIC

Share what you are learning — even a single prompt result. "Here is what I tried, here is what I got" is valuable content. TikTok / LinkedIn / Instagram. High ROI.

🏗️

4. CONTEXT ENGINEERING

Generic prompt = generic answer. Add your role, context, constraints, and ask for clarifying questions. Formula: Richer context = Expert-level output.

🥊

5. SPARRING PARTNER

Stop using AI as a search engine. Use it as a coach, critic, and devil's advocate. Find risks, test assumptions, build better ideas through friction.

💻

6. VIBE CODING

Build one simple app to start. Use the cheapest tools first. You do not need to code — just direct the AI clearly.

⚙️

7. AI SYSTEM

Build systems, not just prompts. Example: Customer support ticket system — AI listens, queries knowledge base, drafts response. Connect to CRM, billing, booking.

📚

8. DOCUMENTATION

Your AI brain = your documentation. What does your business do? What products? What problems? Keep your docs updated — it compounds over time. Formula: AI Leverage = Skill × Clarity.

💻 Slash Commands

Type these at any point inside a Claude Code terminal session.

Session Control

  • /goalSet a stop hook — session won't end until condition is met
  • /rewindRewind the conversation to an earlier point
  • /loopAutonomous loop mode — Claude self-paces a repeating task
  • /compactCompress conversation to free up context window mid-session
  • /clearClear conversation history and start fresh

Memory

  • /memoryOpen and edit your CLAUDE.md directly
  • /rememberSave something to memory right now
  • /initCreate a CLAUDE.md file in the current project

Code Review

  • /code-reviewReview current branch — add ultra for multi-agent cloud review
  • /reviewLightweight code review alias
  • /ultrareviewDeprecated — use /code-review ultra instead

Model & Speed

  • /modelSwitch AI model — Opus / Sonnet / Haiku / Fable
  • /fastToggle Opus fast mode — same model, faster output
  • /costShow token usage and cost for this session

Settings & Health

  • /configOpen settings and configuration panel
  • /permissionsManage tool and file allowlists
  • /doctorDiagnose Claude Code installation health
  • /statuslineStatus line display settings

Account & Info

  • /statusShow account, subscription tier, and usage
  • /terminal-setupConfigure shift+enter and other key bindings
  • /vimToggle vim keybinding mode
  • /loginSwitch accounts
  • /logoutLog out of Claude Code
  • /release-notesView what changed in recent updates
  • /feedbackSubmit feedback to Anthropic
  • /helpGet help and see all available commands
